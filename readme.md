# 🚀 Arch Post Install

A simple post-install script that turns a fresh **Arch Linux + Hyprland** installation into **my complete daily setup** with a single command.

> Reproducible, fast, and opinionated.

## ✨ Features

- 📦 Installs **yay** (AUR helper)
- 🌍 Updates the fastest Pacman mirrors
- 🔧 Installs essential packages and suggests optional ones
- 📁 Fixes Dolphin MIME associations and default file handlers
- 🐚 Installs **Ambxst Shell**
- 🎨 Restores all my dotfiles using **GNU Stow**
- 🎥 Installs my **YTD** YouTube downloader script
- 🧹 Performs final system cleanup:
  - Removes unnecessary packages
  - Restores my `mkinitcpio.conf`
  - Applies my bootloader configuration
  - Installs SDDM themes and overrides
  - Configures required kernel modules
  - Restores `.face.icon`

## 🎯 Goal

The goal of this project is simple:

> Make my entire Arch Linux setup reproducible with one script after a fresh installation.

## ⚠️ Note

This project is **highly opinionated** and tailored to **my personal workflow**.  
Feel free to fork it and customize it for your own setup.
